# QA Fullstack Playwright Framework

Enterprise-grade automation framework for UI and API testing using Playwright, with GitLab CI/CD and Docker support.

## 🚀 Features

- **Unified Testing**: Both UI and API tests in one framework
- **Page Object Model**: Clean separation of test logic and page interactions
- **TypeScript**: Full type safety and better IDE support
- **CI/CD Ready**: GitLab CI pipeline configuration included
- **Docker Support**: Containerized execution for consistency
- **Parallel Execution**: Faster test runs with Playwright's parallel capabilities
- **Comprehensive Reporting**: HTML reports with screenshots and videos

## 📋 Prerequisites

- Node.js 18+
- npm or yarn
- Docker (optional)
